== Description ==

With this plugin you can create banners, menus,headers and photo galleries from the images in a folder.
If the plugin does not work, try downloading from http://www.hotwordpressplugins.com
		

== Installation ==



1. Upload `banner-design` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [`banner_design` ID] in your Posts, where ID is the Banner Design ID.
4. Configure and create new image galleries in your Wordpress Admin: Settings->Banner Design
5. Main Admin params:

	- Images Folder. Folder where the images. Upload the images you want to show on the carousel (jpg, png or gif) to the folder you specify in this parameter. The correct format of the folder would be: folder/, the default is wp-content/plugins/banner-design/images/
	- Timeback. Transition between images, in seconds.
	- Flash Width. Number indicating the width of the gallery. You can use a number in pixels or percentage. For example: 100% or 400px.
	- Flash Height. Number indicating the height of the gallery. You must use a number with px. For example 400px.
	- Images link. Images link separated by Backspace. The first line corresponds to the first image, the second line to the second image and so on. The images are sorted alphabetically.
	- Images titles. Images titles separated by Backspace. The first line corresponds to the first image, the second line to the second image and so on. The images are sorted alphabetically.

== Frequently Asked Questions ==

Once installed and activated, and the default plugin works correctly. The parameters are common to change the folder where you write the server folder containing the images.
